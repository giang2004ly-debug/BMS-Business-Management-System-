# BMS-Business-Management-System-
BMS (Business Management System) là hệ thống quản lý kinh doanh hỗ trợ quản lý sản phẩm, kế toán và nhân sự. Dự án tập trung vào việc thiết kế test case và báo cáo lỗi cho chức năng Create Order / Order Management, nhằm đảm bảo quá trình tạo và quản lý đơn hàng hoạt động chính xác.
